# Portfolio Website — Yaramala Sai Sandeep Reddy

A premium personal portfolio website with aurora gradient theme,
glassmorphism UI, and smooth animations.

## Tech Stack
- HTML5, CSS3, JavaScript (Vanilla)
- Animations: Canvas API, IntersectionObserver, CSS keyframes

## Features
- Aurora animated background
- Mouse parallax hero section
- Scroll-triggered reveal animations
- Responsive across all devices

## Setup
1. Clone the repository
2. Open index.html in a browser
   or use Live Server in VS Code
